关于蜂窝状图片的实现

发表于 2018-03-29 | 分类于 css
论实现蜂窝状图片墙的几种方法

使用三个元素实现六边形效果

  • 思路：使用三个同样大小的div相互叠加，形成正六边形区域
  • 优点：兼容性最好
  • 缺点：结构繁琐，难以扩展

准备三个同样大小的div，其中一个逆时针旋转60deg,另一个顺时针旋转60deg,三个div叠加的部分刚好是正六边形。

使用一个元素配合css变形得到六边形

  • 思路：利用旋转、斜切和位移达到六边形效果
  • 优点：方便扩展，结构清晰
  • 缺点：理解困难，曲线陡峭

通过css切片实现

  • 思路：利用css切片实现六边形
  • 优点：方便扩展，结构简单
  • 缺点：兼容性差，学习曲线陡峭
